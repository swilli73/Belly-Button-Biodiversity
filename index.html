Belly Button Biodiversity Dashboard

Use the interactive charts below to explore the dataset

The purpose of this project is to visualize a dataset containing species of bacteria contained in the navels of several test subjects. When picking a subject, their demographic info, top 10 bacteria cultures found, belly button washing frequency, and number of bacteria cultures found per sample are displayed. This data is being used for the purpose of using these species of bacteria to manufacture synthetic beef.
